# Aplicativo de Mapeamento Arqueológico

## Visão Geral

O trabalho arqueológico em campo exige precisão, organização e registro rápido das informações coletadas. Porém, muitos arqueólogos lidam com ferramentas dispersas, blocos de notas, planilhas externas e imagens sem marcações, o que pode dificultar a documentação eficiente dos sítios.

Nosso projeto propõe um aplicativo mobile desenvolvido em Kotlin e Java que permite mapear sítios arqueológicos diretamente a partir de imagens, adicionar pontos-zero, registrar artefatos descobertos e organizar seus detalhes de forma clara e acessível.

## Objetivo

- Simplificar o mapeamento de sítios arqueológicos por meio de imagens anexadas
- Organizar artefatos em banco de dados local, com informações estruturadas
- Estabelecer pontos-zero para referência espacial dentro da própria foto

## Impacto Esperado

- **Documentação mais rápida e organizada**, reduzindo o tempo de trabalho em campo
- **Maior precisão** ao utilizar imagens anotadas e pontos-zero
- **Facilitação da análise posterior**, com dados bem estruturados
- **Ferramenta offline**, por conta de baixa possibilidade de wi-fi em sítios arqueológicos

## Tecnologias Utilizadas

- Kotlin + Java
- Android Studio
- Room Database
